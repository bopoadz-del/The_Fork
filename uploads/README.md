# Upload Real Construction Files Here

This directory is gitignored — files dropped here stay local.

## Layout

- contracts/   — PDF/DOCX contract documents
- drawings/    — PDF architectural / engineering drawings; DXF for QTO
- schedules/   — Primavera P6 (.xer, .xml), MS Project (.mpp), Excel
- bim/         — IFC, Revit exports, point clouds
- photos/      — Site / progress / inspection photos (jpg, png)
- reports/     — Progress, test, inspection reports (PDF, DOCX)

## Run tests

    ./test_real_files.sh           # auto-iterates each populated dir
    python3 test_single_file.py uploads/contracts/x.pdf
    ./diagnose.sh                  # block / lib / file-access health

Synthetic fixtures live under uploads/_synthetic/ — generated by
scripts/generate_test_fixtures.py for end-to-end pipeline plumbing
verification when no real files are present.
