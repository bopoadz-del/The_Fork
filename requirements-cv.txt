# Optional dependencies for the local CV tier (PR 3b of the ML roadmap).
#
# Install only when you want object detection on site photos via a local
# YOLO model:
#
#     pip install -r requirements-cv.txt
#
# When NOT installed, the platform behaves exactly as before: the image
# block's metadata + OCR paths work, the `detect_objects` operation
# returns ``{available: false}``, and the `analyze` path quietly skips
# the detection section. No code paths break.
#
# Disk + cache cost:
# - YOLOv8n weights are ~6 MB, cached to ~/.config/Ultralytics/ on first
#   use. Downloaded from the ultralytics CDN, not Hugging Face.
# - torch (~250 MB) is pulled in transitively — the heaviest cost.
#
# Note on class coverage: the base YOLOv8n is COCO-trained (80 classes
# including 'person', 'car', 'truck'). Construction-specific PPE
# detection (hard-hat / safety-vest / harness) is a custom-trained model
# that ships separately — see docs/EDGE_PORT.md for the fine-tune recipe
# when you want PPE-aware detection.

ultralytics>=8.0.0
